Setup & Deployment Guide

Step 1: File Structure

Create a folder with the following files:

gulder-website/ ├── index.html ├── styles.css └── script.js

Step 2: Google Sheets Integration

To save registration data to Google Sheets:

  1. Create a new Google Sheet
  2. Add columns: Name, Phone, State, LGA, Referral Code, Registration Date
  3. Go to Tools > Script Editor
  4. Paste this code:
function doPost(e) { const sheet = SpreadsheetApp.getActiveSheet(); const data = JSON.parse(e.postData.contents); sheet.appendRow([ data.name, data.phone, data.state, data.lga, data.referralCode, data.registeredDate ]); return ContentService.createTextOutput(JSON.stringify({status: 'success'})) .setMimeType(ContentService.MimeType.JSON); }
  • Deploy as web app (Execute as: Me, Anyone can access)
  • Copy the deployment URL and update in script.js line ~305

    • Step 3: Update Script.js

    Replace the Google Sheets URL in the sendToGoogleSheet function:

    fetch('YOUR_GOOGLE_APPS_SCRIPT_URL', { method: 'POST', body: JSON.stringify(userData) })

    Step 4: Banner Images

    Replace the placeholder image URLs with your own:

    • Top Banner: https://via.placeholder.com/1200x300?text=GULDER+ULTIMATE+RETURNS
    • Bottom Banner: https://via.placeholder.com/1200x300?text=JOIN+THE+QUEST
    • Energy Box: https://via.placeholder.com/400x300?text=GULDER+ENERGY

    You can upload your images to a CDN or use data URLs.

    Step 5: Deploy to Hosting

    Options:

    • GitHub Pages: Push files to GitHub repo, enable Pages
    • Netlify: Drag and drop folder or connect GitHub
    • Vercel: Similar to Netlify
    • Firebase Hosting: Use Firebase CLI
